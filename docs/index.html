TieBreaker AI

Prédictions intelligentes de matchs de tennis ATP — modèles Elo par surface, données historiques complètes et une CLI simple à utiliser.

Basé sur des jeux de données ATP (1968–2024). Conçu pour être extensible et reproductible.

Installer En savoir plus

Essayer localement

python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
./executable/build
./TieBreaker --help

Fonctionnalités

Données historiques

Accès à plus de 50 ans de matchs ATP, y compris challengers et qualifications.

ELO par surface

Classements dynamiques sensibles à la surface (terre, gazon, dur, indoor).

CLI pratique

Interrogations rapides de classements, confrontations, et filtres avancés.

Extensible

Base pour intégrer modèles ML et nouveaux jeux de données.

Installation

  1. Cloner le dépôt : git clone https://github.com/ligsow6/TieBreakAI.git
  2. Créer un environnement : python3 -m venv .venv
  3. Activer et installer : source .venv/bin/activate && pip install -r requirements.txt
  4. Générer le lanceur : ./executable/build

Utilisation

Exemples rapides :

  • ./TieBreaker rank --player "Novak Djokovic"
  • ./TieBreaker match --p1 "Rafael Nadal" --p2 "Novak Djokovic" --surface Clay

Options globales : --data-root PATH, --help.
